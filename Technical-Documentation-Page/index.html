Introduction

Python is a popular, high-level programming language known for its readability and flexibility.

It supports multiple paradigms including object-oriented, procedural, and functional programming.

print("Hello, world!")
  • Easy to learn
  • Widely used in industry
  • Open source
Variables and Data Types

Variables in Python do not need explicit declaration to reserve memory space.

The declaration happens automatically when you assign a value to a variable.

x = 5 name = "Alice"
  • int
  • float
  • str
  • bool
Control Flow

Python provides various control flow tools such as if statements, loops, and more.

Indentation is used to define blocks of code instead of curly braces.

if x > 10:
  print("Greater than 10")
  • if/elif/else
  • while loops
  • for loops
Functions

Functions are defined using the `def` keyword.

They help in code reusability and modular programming.

def greet(name):
  return f"Hello, {name}"

Functions can take arguments and return results.

Modules and Packages

Modules allow you to organize your Python code across multiple files.

Packages are directories of modules with a special `__init__.py` file.

import math from mypackage import mymodule
  • Standard Library
  • Third-party modules
  • Custom modules
